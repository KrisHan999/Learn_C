

## switch



### switch 结构：

```
switch(表达式):{
    case 常量1：
        语句1;
        break;
    case 常量2:
        语句2;
        break;
    ....
    default:
        语句;
}
```





**使用switch结构需要注意的点：**

1. **switch后的表达式只能是整形或字符型，不支持字符串**

2. case后的常量表达式不能相同

3. case后允许多条语句，不需要大括号

4. **如果不添加break语句，需要特别注意执行顺序**

5. case和default字句的先后顺序可以自行变动

6. default子句可以省略
